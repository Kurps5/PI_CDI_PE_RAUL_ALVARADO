# Proyecto Integral de Desarrollo y Validación de Software en Lenguaje C

## Descripción

Este proyecto consiste en el desarrollo de un Sistema de Gestión de Inventario Comercial utilizando Programación Estructurada en lenguaje C.

## Funcionalidades

- Registrar productos
- Mostrar productos
- Buscar productos
- Modificar productos
- Eliminar productos
- Validación de datos

## Tecnologías utilizadas

- Lenguaje C
- Code::Blocks
- Programación Estructurada

## Estructura del proyecto

```
main.c
inventario.c
inventario.h
validaciones.c
validaciones.h
```
